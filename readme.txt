ask_for_input function:
    Takes an input from the user and converts it into lowercase and then checks if it is character
    if not a character, write out a  message and allow the user to take another guess. 

check_guess() function:
    Looks for the first occurence of the letter within the chosen random word
    if there is a match, place the character in to the display
    if there is no match, the count is incremented
